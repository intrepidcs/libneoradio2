Installation

neoradio2 publishes pre-built wheels for Windows, macOS, and Linux (CPython 3.9–3.14), so in most cases installation is simply:

pip install neoradio2

**Windows:** ``pip.exe`` is usually located under the ``Scripts`` directory
of your Python installation.

**Linux:** the udev rules must be installed so devices are accessible as a
normal user. Copy `99-intrepidcs.rules
<https://github.com/intrepidcs/libneoradio2/blob/master/libneoradio2/99-intrepidcs.rules>`_
to ``/etc/udev/rules.d/`` and run
``sudo udevadm control --reload-rules && sudo udevadm trigger``.

Building from source

If no wheel is available for your platform, pip builds from source. The repository uses git submodules, so a source build needs a CMake toolchain and a compiler (MSVC on Windows, GCC/Clang elsewhere). See the README for details.
